# Mental Health in Newfoundland and Labrador

This notebook studies a data set on self-reported mental health of Canadian citizens, published
online by the Government of Canada. The data set is available for download [here.](https://open.canada.ca/data/en/dataset/480a161e-56c6-4a86-869c-e38381b329a5)
The purpose of this notebook is to determine how citizens of Newfoundland and Labrador rate their own mental health.
